# Text Formatting

Bold

```md
**Bold**
```

**Bold**

Italic

```md
_Italic_
```

_Italic_

Bold + Italic

```md
**_Bold Italic_**
```

**_Bold Italic_**

Strikethrough

```md
~~Deleted~~
```

~~Deleted~~

Inline code

```md
`git status`
```

`git status`

# Lists

Unordered

```md
- HTML
- CSS
- JavaScript
```

Ordered

```md
1. Clone repository
2. Install packages
3. Run project
```

Nested

```md
- Frontend
  - HTML
  - CSS
  - React
```
