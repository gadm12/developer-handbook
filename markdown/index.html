| Command | Purpose |
|---|---|

GitHub Alerts

TIP

IMPORTANT

WARNING

Something that may cause a problem or data loss.

Syntax

> Useful information.

> [!TIP]
> Helpful advice.

> [!IMPORTANT]
> Information required for success.

> [!WARNING]
> Something may go wrong.

> [!CAUTION]
> Be careful before continuing.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/username/project.git
   ```

2. Move into the project:

   ```bash
   cd project
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the project:

   ```bash
   npm run dev
   ```
