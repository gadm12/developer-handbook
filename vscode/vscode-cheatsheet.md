# Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+P | Quick Open |
| Ctrl+Shift+P | Command Palette |
| Ctrl+` | Toggle Terminal |
| Ctrl+/ | Comment line |
| Ctrl+D | Select next occurrence |
| Alt+Up | Move line up |
| Alt+Down | Move line down |
| Shift+Alt+Down | Duplicate line |
| F2 | Rename symbol |
| Ctrl+Shift+L | Select all occurrences |
| Ctrl+Space | Trigger IntelliSense |
