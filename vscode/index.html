Terminal

Terminal Controls

Ctrl+` Open or hide the terminal panel
+ Create another terminal
Trash Close the selected terminal

Terminal Commands

clear Clear terminal output
Ctrl+C Stop a running command
code . Open the current folder in VS Code
Ctrl+C inside the terminal stops a running program. In the editor, it normally copies selected text.

Find and Replace

Search

Current File

Ctrl+F Search in the current file
Ctrl+H Replace in the current file
Enter Move to the next match
Shift+Enter Move to the previous match

Entire Project

Ctrl+Shift+F Search every file in the project
Ctrl+Shift+H Replace matches across the project
Project search supports plain text, case-sensitive matching, whole words, and regular expressions.

Example Settings JSON

{
```

"editor.fontFamily": "Fira Code",
"editor.fontLigatures": true,
"editor.formatOnSave": true,
"editor.defaultFormatter": "esbenp.prettier-vscode",
"editor.lineNumbers": "relative",
"editor.wordWrap": "off",
"workbench.colorTheme": "One Dark Pro Flat Night"
}
```
A project can override your default formatter with workspace settings or language-specific settings.
