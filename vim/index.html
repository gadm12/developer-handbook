Vim States

Modes

Esc

Normal Mode

Navigate the file and run editing commands.

i

Insert Mode

Type regular text into the file.

v

Visual Mode

Select characters and text regions.

V

Visual Line Mode

Select full lines.

Ctrl+v

Visual Block Mode

Select rectangular columns of text.

When a command behaves unexpectedly, press Esc to return to Normal mode.

Navigate the File

Movement

File Movement

gg Beginning of file
G End of file
25G Go to line 25

Line Movement

0 Beginning of line
^ First non-space character
$ End of line

Word Movement

w Beginning of next word
b Beginning of previous word
e End of word

Vertical Movement

j Move down
k Move up

5k Move up 5 lines

Page Movement

Ctrl+d Move half a page down
Ctrl+u Move half a page up

Modify Text

Editing

Enter Insert Mode

i Insert before cursor
I Insert at beginning of line
a Append after cursor
A Append at end of line

Create New Lines

o New line below
O New line above

Delete

x Delete character
dd Delete line
dw Delete word forward
d$ Delete to end of line
diw Delete inside word

Change

ciw Change inside word
ci( Change inside parentheses
ci" Change inside quotes
cc Change current line
A change command deletes the selected text and immediately places you in Insert mode.

Yank and Put

Copy and Paste

Copy

yy Copy current line
yiw Copy inside word
yw Copy to next word
y$ Copy to end of line
yap Copy paragraph

Paste

p Paste after cursor or line
P Paste before cursor or line
"0p Paste your last yank
Vim calls copying yanking. Deleting text also replaces the default paste register, so "0p is useful when you want your previous yank.

Reverse Changes

Undo and Redo

u

Undo

Reverse the most recent change.

Ctrl+r

Redo

Restore a change that was undone.

Select Text

Visual Mode

Selection Modes

v Character selection
V Full-line selection
Ctrl+v Block or column selection

Edit a Selection

y Copy selection
d Delete selection
c Change selection
> Indent selection
< Remove indentation

Find Text

Searching

/pokemon Search forward for “pokemon”
?pokemon Search backward for “pokemon”
n Next match in search direction
N Previous match
:noh Clear search highlighting

Substitute Text

Replace

Small Replacements

r Replace one character
ciw Replace the current word
ci" Replace text inside quotes

File-Wide Replace

:%s/old/new/g Replace every match
:%s/old/new/gc Confirm each replacement
:s/old/new/g Replace matches on current line
Use the c confirmation flag when you are not certain that every match should be replaced.

Save and Exit

Files

:w Save file
:q Quit
:wq Save and quit
:x Save if changed, then quit
:q! Quit without saving
