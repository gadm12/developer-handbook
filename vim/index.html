Highest Priority

Daily Vim Workflows

These cards group the Vim motions and commands developers usually use together: move precisely, edit quickly, search safely, manage files, and stay productive inside VS Code.

Core Editing

Navigate and Edit

Normal Mode 
Esc
gg
/target
n
ciw
Esc
:w

Return to Normal mode, jump into the file, search for the next match, change the word under the cursor, and save without leaving the keyboard.

Text Moves

Copy, Delete, and Paste

Yank 
yy
p
V
j
d
"0p

Copy a line, paste it nearby, expand to a visual selection, delete the selection, and restore your most recent yank instead of the deleted text.

Refactor

Search and Replace

Substitute 
/oldName
n
ciwnewName
Esc
:%s/oldName/newName/gc
:noh

Find the target text, update the current match quickly, then run a file-wide confirmed replacement and clear highlights when you finish.

Files

Work with Multiple Files

Buffers 
:e app.py
:vsplit utils.py
:ls
:bn
:w
:q

Open another file, split the view, inspect your buffers, move to the next one, save your work, and quit cleanly.

Vim States

Modes

Esc

Normal Mode

Navigate the file and run editing commands.

i

Insert Mode

Type regular text into the file.

v

Visual Mode

Select characters and text regions.

V

Visual Line Mode

Select full lines.

Ctrl+v

Visual Block Mode

Select rectangular columns of text.

When a command behaves unexpectedly, press Esc to return to Normal mode.

Navigate the File

Movement

File Movement

ggBeginning of file
GEnd of file
25GGo to line 25

Line Movement

0Beginning of line
^First non-space character
$End of line

Word Movement

wBeginning of next word
bBeginning of previous word
eEnd of word

Vertical Movement

jMove down
kMove up
10jMove down 10 lines
5kMove up 5 lines

Page Movement

Ctrl+dMove half a page down
Ctrl+uMove half a page up

Modify Text

Editing

Enter Insert Mode

iInsert before cursor
IInsert at beginning of line
aAppend after cursor
AAppend at end of line

Create New Lines

oNew line below
ONew line above

Delete

xDelete character
ddDelete line
dwDelete word forward
d$Delete to end of line
diwDelete inside word

Change

ciwChange inside word
ci(Change inside parentheses
ci"Change inside quotes
ccChange current line
A change command deletes the selected text and immediately places you in Insert mode.

Yank and Put

Copy and Paste

Copy

yyCopy current line
yiwCopy inside word
ywCopy to next word
y$Copy to end of line
yapCopy paragraph

Paste

pPaste after cursor or line
PPaste before cursor or line
"0pPaste your last yank
Vim calls copying yanking. Deleting text also replaces the default paste register, so "0p is useful when you want your previous yank.

Reverse Changes

Undo and Redo

u

Undo

Reverse the most recent change.

Ctrl+r

Redo

Restore a change that was undone.

Select Text

Visual Mode

Selection Modes

vCharacter selection
VFull-line selection
Ctrl+vBlock or column selection

Edit a Selection

yCopy selection
dDelete selection
cChange selection
>Indent selection

Find Text

Searching

/pokemonSearch forward for "pokemon"
?pokemonSearch backward for "pokemon"
nNext match in search direction
NPrevious match
:nohClear search highlighting

Substitute Text

Replace

Small Replacements

rReplace one character
ciwReplace the current word
ci"Replace text inside quotes

File-Wide Replace

:%s/old/new/gReplace every match
:%s/old/new/gcConfirm each replacement
:s/old/new/gReplace matches on current line
Use the c confirmation flag when you are not certain that every match should be replaced.

Save and Exit

Files

:wSave file
:qQuit
:wqSave and quit
:xSave if changed, then quit
:q!Quit without saving

Extension Workflow

VS Code Vim

VS Code Shortcuts

Ctrl+Shift+POpen Command Palette

Leader Shortcuts

<Space> fOpen a file
<Space> sSave current file
<Space> eOpen Explorer
Your leader key is Space, so leader shortcuts begin by pressing the spacebar in Normal mode.
