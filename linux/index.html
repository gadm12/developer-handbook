Command Line

Linux Terminal Cheat Sheet

The fastest reference for moving around a project, managing files, searching code, checking processes, and handling the Linux commands you use while developing every day.

Coreutils Docs →

Highest Priority

Daily Linux Workflows

These cards group the terminal commands developers usually need as one task: open the project, search for code, manage files, stop stuck processes, and install packages safely.

Start Work

Open and Inspect a Project Folder

Navigate 
cd ~/code/project
pwd
ls -la
git status
code .

Use this sequence when you first drop into a project and want to confirm the folder, inspect files, check Git state, and open the workspace in VS Code.

Search

Find Files and Search Text

Inspect 
find . -name "*.py"
grep -rn "useState" .
grep -ri "pokemon" .
less path/to/file

This is the practical search flow for locating files, finding text recursively, ignoring letter case when needed, and opening a file to inspect it without editing.

File Work

Create, Move, and Clean Up Files

Files 
mkdir -p src/components
touch src/components/App.jsx
cp src/components/App.jsx src/components/App.backup.jsx
mv src/components/App.backup.jsx src/components/App.old.jsx
rm src/components/App.old.jsx

Group these together when you are creating folders, making quick file copies, renaming files, and cleaning up temporary artifacts during development.

Troubleshooting

Check and Stop a Running Process

Processes 
ps aux | grep python
kill PID
kill -9 PID
Ctrl + C

Start with a normal stop, use a force kill only when required, and remember that Ctrl + C is the fastest way to stop the foreground server you just started.

System Setup

Install and Update a Package

APT 
sudo apt update
sudo apt upgrade
sudo apt install package-name
sudo apt remove package-name

This is the normal Linux package-management sequence: refresh package information, apply upgrades, install what you need, and remove software you no longer want.

Move Around

Navigation

Current Directory

pwd

Prints the full path of your current location.

List Files

ls

Displays files and folders in the current directory.

Detailed List

ls -l

Shows permissions, owner, size, and modification date.

Show Hidden Files

ls -la

Includes hidden files such as .gitignore.

Change Directory

cd folder

Moves into the selected directory.

Go Back One Folder

cd ..

Moves to the parent directory.

Go Home

cd ~

Moves to your Linux home directory.

Previous Directory

cd -

Switches between your current and previous directory.

Use pwd whenever you are unsure which folder your terminal is currently using.

File Management

Files and Directories

Directories

Create Folders

mkdir project

Create one directory.

mkdir -p src/components

Create nested directories, including missing parent folders.

Files

Create a File

touch app.py

Create an empty file.

Copy a File

cp old.py new.py

Copy a Directory

cp -r folder backup

The -r option copies the folder and everything inside it.

Move or Rename

mv old.py new.py

The same command can rename a file or move it to another folder.

Use Carefully

Delete Commands

Delete a File

rm file.py

Delete a Folder

rm -r folder

Delete an Empty Folder

rmdir folder

Terminal deletions do not normally go to the Recycle Bin.

Read Content

Viewing Files

Display Entire File

cat file.txt

Prints the full file directly in the terminal.

First 10 Lines

head file.txt

Displays the beginning of a file.

Last 10 Lines

tail file.txt

Displays the end of a file.

Follow a Log File

tail -f logfile.log

Shows new log entries as they are added.

View Page by Page

less file.txt

Opens a scrollable file viewer.

Exit Less

q

Closes the less file viewer.

Find Things

Searching

Files

Find by Name

find . -name "*.py"

Search the current directory and its subdirectories for Python files.

Text

Search Inside a File

grep "Pokemon" app.py

Find lines containing the selected text.

Search Recursively

grep -r "useState" .

Search every file under the current directory.

Case-Insensitive Search

grep -ri "pokemon" .

The -i option ignores uppercase and lowercase differences.

Show Line Numbers

grep -rn "useState" .

Access Control

Permissions

View

Check Permissions

ls -l

Shows permissions at the beginning of each file's row.

Execute

Make a Script Executable

chmod +x script.sh

Adds permission to run the file as a program.

Run an Executable Script

./script.sh
A shell script may need both a proper shebang and executable permission before it can be run with ./script.sh.

Development Helpers

Git and Editor Helpers

Git

Current Branch

git branch

Lists local branches and marks the current branch with an asterisk.

VS Code

Open Current Folder

code .

Opens the current terminal directory as a VS Code workspace.

Check Git Status

git status

Open One File in VS Code

code app.py

Running Programs

Processes

Show Current Processes

ps

Detailed Process List

ps aux

Find a Process

ps aux | grep python

Stop

Kill a Process

kill PID

Sends a normal termination request to the process.

Force

Force Kill

kill -9 PID

Immediately stops a process that will not close normally.

Try kill PID before using kill -9 PID.

APT Package Manager

Install and Remove Software

Update Package Information

sudo apt update

Downloads the latest available package information.

Upgrade Packages

sudo apt upgrade

Installs available updates for existing packages.

Install a Package

sudo apt install package-name

Downloads and installs a new package.

Remove a Package

sudo apt remove package-name

Uninstalls the selected package.

Network Tools

Networking

Show IP Information

ip addr

Displays network interfaces and IP addresses.

Test a Connection

ping google.com

Tests whether a host can be reached over the network.

Download a File

wget URL

Downloads a file from a URL.

Stop a continuously running ping command with Ctrl + C.

Keyboard

Useful Terminal Shortcuts

Ctrl + C

Stop Command

Stop a running command, server, or program.

Ctrl + Z

Suspend Program

Pause the current foreground process.

Ctrl + L

Clear Screen

Clear visible terminal output without deleting command history.

Ctrl + R

Search History

Search for a command you previously entered.

Tab

Auto-Complete

Complete file names, folder names, and some commands.

↑

Previous Command

Move backward through command history.

Clear the Terminal

clear

Display Command History

history
